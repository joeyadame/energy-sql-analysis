# energy-sql-analysis
SQL analysis of U.S. energy data (EIA)

This repository contains SQL queries used to analyze historical U.S. energy data stored in PostgresSQL.

## Purpose
The goal of this project is to gain experince writing SQL queries on real-world energy data.
